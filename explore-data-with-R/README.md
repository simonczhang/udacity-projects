### Introduction
In this project, I explore the Red Wine data set use exploratory data analysis techniques in R to find relationships in one variable,  multiple variables, and also examine the data for distributions, outliers, and anomalies.

This project is like a stream-of-consciousness analysis as I ask questions, create visualizations, and explore the data.
